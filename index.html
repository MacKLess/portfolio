Current Projects

The following are my previous projects, for your viewing convenience. Author recommendations are indicated by the green text. Enjoy!

  • hello-world This was my first webpage I cowrote with Kristen Kulha listing forms of greeting in various languages around the world. Thrilling stuff!

  • fluffy-thorfeld-adventures Co-written with Rafael Furry, this was my first implementation of a README page as well as linked/nested pages, detailing the adventure of two of my pets: Fluffy the Python and Thorfeld the Bulldog.
  • last-resort Fun project with codeveloper Furry, incorporating music, linked media, photos, and branching versions of a lovely little dystopic resort off the coast of Japan. The last vacation you'll ever want to take!
  • cupcake-shop Three page webpage created with Erik Zakrzewski for our "Cupcake Shack." Fun experiment with photos, background colors, and styling with classes.
  • boring-lecture Another styling with classes website collaboration with Zakrzewski. Updated later in the day with div classes.
  • blogging Exercise focusing on styling with selectors, designed with Zakrewski. This was our first use of an "About Me" section.
  • gwar Zakrewski and I came up with the rather amusingly titled "Gwe are Gwar" fan website in which we practiced floats using only Git, CSS, and HTML.
  • renovations Was created (again with Zakrewski) with Git, HTML, and CSS to practice using columns. Our subject was kitchen renovations with a column each for: before, during, and after.

  • week-in-review Elrey Belmonti and I created this webpage using text supplied by Epicodus and using our first front-end framework, Bootstrap. What a revelation! We thoroughly enjoyed exploring the different possibilities with the templates and the speed with which we accomplished polished effects with considerable ease and speed compared with the limited use of only css and html. Consider me a convert.
  • vacation An enjoyable page codesigned with Belmonti extolling the virtues of vacationing in Japan during Cherry Blossom season. It worked --now I want to go! In this site, we heavily utilized Bootstrap, CSS, HTML and Git.
  • our-wiki Saving the best for last, Belmonti and I used Bootstrap, CSS, HTML, and Git to recreate a Wikipedia page. We made free use of columns, rows, more div classes than ever before, and our first experience building tabs. The end result is not identical to the modelled page, but gratifyingly similar. We hope you enjoy it!
