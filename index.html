Featured Projects

Cherry Blossom Dreaming

Japan: Cherry Blossom Dreams was created to promote the various benefits of vacationing in Japan during the Cherry Blossom season. Three types of vacation packages are highlighted utilizing float and box models. The site features Git, HTML, CSS, and Bootstrap and was created in collaboration with E. Belmonti.

Wiki Clone


Wiki Clone is an attempt to recreate the Wikipedia mainpage on 8/3/17. The page changes everyday, so the cloned page and the Wikipedia link will have content changes (text, images, and links), but you can see several elements that were recreated in this version. Co-created with E. Belmonti using Git, HTML, CSS, and Bootstrap.

Last Resort

Last Resort is a dystopic vacation resort located on the derelict Hashima Island. Designed as one of three potential promotional angles using Git branch and merge functions, the site boasts a Gabber musical background and a video about this type of electronic music, a subgenre of hardcore techno. Designed in collaboration with R. Furry using Git, HTML, and CSS.
